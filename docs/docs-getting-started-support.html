Support & Updates

Welcome to UltraViolet Pro support! We're Huement - a newly formed software studio passionate about creating high-quality admin dashboards and development tools. Thank you so much for your purchase and for supporting us as we grow!

What's Included

Your purchase includes 6 months of guaranteed ongoing support from the date of purchase. We're committed to helping you succeed with UltraViolet Pro.

Our Support Commitment

  • Bug fixes and issue resolution
  • Technical assistance with setup and configuration
  • Help with component usage and customization
  • Compatibility updates for dependencies
  • Security patches
  • Bi-monthly updates with improvements and new features

Regular Updates

We publish bi-monthly updates (every two months) that include:

  • New components and features
  • Performance improvements
  • Bug fixes and patches
  • Documentation updates
  • Compatibility updates for Laravel, Livewire, and other dependencies

Updates are free during your support period. Check the changelog for details on what's included in each release.

Getting Help

1. Documentation First

Before reaching out, please check:

  • This documentation (you're reading it now!)
  • Code comments in the source files
  • Component examples in the dashboard

Most common questions are answered in the documentation.

2. Contact Us Directly

Need personalized assistance? We're here to help!

Visit: huement.com

Use the contact form on our website to reach us directly. We typically respond within 24-48 hours on business days (Monday-Friday).

3. GitHub Issue Tracking

We use GitHub for bug tracking and issue management:

  • UltraViolet Pro (Private Repository): Your purchase includes access to our private GitHub repository for issue tracking. Contact us to get access credentials.
  • Free Version (Public Repository): We also maintain a free/public version of UltraViolet with a public GitHub repository that we use for tracking common issues that affect both versions.

Best for:

  • Reporting bugs
  • Technical issues
  • Installation problems
  • Compatibility concerns

When submitting an issue, please include:

  • UltraViolet version number
  • Laravel version
  • PHP version
  • Node/npm version
  • Browser and OS information
  • Steps to reproduce the issue
  • Screenshots or error messages (if applicable)

Feature Requests

Have an idea for a new component, layout, or feature? We want to hear it!

Your feedback directly influences our development priorities. Many of the features in UltraViolet Pro came from user suggestionswe're building this for you, with you.

How to Submit Feature Requests

Submit via: huement.com contact form or GitHub issues

Please include:

  • Detailed description of the feature
  • Use case (how you would use it in real projects)
  • Examples, mockups, or references (if applicable)
  • Priority level for your needs (nice-to-have vs critical)

What Happens Next

We review every feature request and prioritize based on:

  1. Community demand - How many users want this?
  2. Technical feasibility - Can we build it well?
  3. Strategic fit - Does it align with UltraViolet's vision?
  4. Implementation complexity - Time and resource requirements

High-demand features often ship within 1-2 release cycles. We're not a slow-moving enterprisewe move fast and ship regularly.

Check the [roadmap]({{ route("docs.show", "roadmap") }}) to see if your requested feature is already planned!

What Support Covers

Covered (Included)

  • Installation and setup assistance
  • Configuration help
  • Bug fixes in core functionality
  • Guidance on using components
  • Compatibility issues with documented dependencies
  • Questions about features and functionality
  • Security vulnerabilities

Not Covered

  • Custom development work
  • Third-party plugin customization (beyond documented integrations)
  • Hosting or server configuration issues
  • General PHP/Laravel/JavaScript training
  • Issues caused by modified core files
  • Conflicts with non-standard environments

For custom development needs, contact us about consulting rates.

Support Guidelines

How to Get the Best Help

  1. Be Specific: Provide detailed information about your issue
  2. Include Context: Share relevant code snippets, screenshots, or error messages
  3. Check First: Review documentation and existing issues before asking
  4. Stay Updated: Keep UltraViolet and dependencies updated to the latest stable versions
  5. Follow Best Practices: Use the recommended setup and configuration

Expected Response Times

  • Critical bugs: 24 hours
  • General support: 24-48 hours on business days
  • Feature requests: Acknowledged within 1 week

Response times may be longer during holidays or peak periods.
