## ğŸ› ï¸ Installation

### Prerequisites
- Web server (Apache, Nginx, or any static file server)
- Modern web browser
- No PHP or database required!

### Quick Start

1. **Download the package**
   - Extract the ZIP file to your web server directory
   - No installation or setup required!

2. **Access the dashboard**
   - Open `index.html` in your web browser
   - Or upload to your web server and visit the URL

3. **Choose your layout**
   - **Vertical Menu**: `index.html` (default)
   - **Horizontal Menu**: `horizontal-dash/index.html`

4. **Customize as needed**
   - Edit HTML files directly
   - Modify CSS in the `css/` directory
   - Update JavaScript in the `js/` directory
